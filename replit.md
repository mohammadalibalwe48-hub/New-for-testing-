# Signup Page

A simple React + TypeScript signup form.

## Stack

- **Frontend**: React 19, TypeScript, Vite 8
- **Styling**: Plain CSS (App.css)

## Project Structure

```
src/
  main.tsx    # React entry point
  App.tsx     # Signup form component
  App.css     # Styles
index.html
vite.config.ts  # Vite config (port 5000, host 0.0.0.0)
```

## Dev

```bash
npm run dev   # starts on port 5000
```

## Deployment

Configured as a static site — builds with `npm run build`, serves from `dist/`.
